# Personal Portfolio

Simple personal portfolio website created to present my profile,
projects and technical skills as a Computer Engineering student.

## Tech Stack

-   Next.js
-   TypeScript
-   TailwindCSS

## Features

-   About me section
-   Projects showcase
-   Skills overview
-   Contact information
-   Responsive design

## Run Locally

``` bash
git clone https://github.com/Alexandre-Ferrete/Personal_Portfolio.git
cd portfolio
npm install
npm run dev
```

Open http://localhost:3000 in your browser.

## Deployment

Recommended deployment: Vercel.
