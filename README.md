# Personal Portfolio

Personal portfolio website created to present my profile, projects and technical skills as a Computer Engineering student.

## Tech Stack

-   Vue.js 3 (Composition API)
-   TypeScript
-   Vite
-   TailwindCSS
-   Pinia (state management)
-   Vue Router

## Features

-   Light/Dark mode toggle
-   About me section with hobbies
-   Projects showcase with hover effects
-   Skills overview by category
-   Professional contact section
-   Responsive design

## Prerequisites

-   Node.js (v18 or higher)
-   npm

## Run Locally

1.  **Clone the repository**

    ```bash
    git clone https://github.com/Alexandre-Ferrete/Personal_Portfolio.git
    ```

2.  **Navigate to the project directory**

    ```bash
    cd Personal_Portfolio
    ```

3.  **Install dependencies**

    ```bash
    npm install
    ```

4.  **Start the development server**

    ```bash
    npm run dev
    ```

5.  **Open in your browser**

    Visit http://localhost:5173

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build |

## Deployment

Recommended deployment: Vercel, Netlify, or any static hosting platform.

To build for production:

```bash
npm run build
```

The output will be in the `dist` folder.
